Atrial Fibrillation Detector

Computer Engineering Graduation Project

Brief:The who idea of the project is to monitor the patient's heart rate and detect any sign of Atrial Fibrillation

General Explanation on how the project works: the bracelet we built will measure the patient's heart rate and send it at the same time to an iOS application we built

This project was built by: Abdulaziz Alharbi, Abdulrahman Abonayan, Rotanak Nou

This video shows the whole project process, from measuring the heart rate using Arduino Beetle and the heart rate sensor, to send the data to the iOS Application and display the data it receives on the applications chart, and lastly, it shows how the application is storing the data it receives in the Firebase database in realtime.

This video shows how the patient can access his/her historical heart rate activities using the iOS Application's calendar
